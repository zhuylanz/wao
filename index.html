Wao API References

Functions

  1. SCAN PAGE:
    • Usage: GET /scanpage
    • Parameters: uri (str), start (unix timestamp), end (unix timestamp), mode (0 | 1 | 2)
    • Returns:
      • mode 0 or none: utime (time of post in unix timestamp), link (link to the post), react (total number of reactions), share (total number of shares), caption (caption of post), comment (total number of comments but only visible comments)
      • mode 1: utime (time of post in unix timestamp), link (link to the post), react (total number of reactions), share (total number of shares), caption (caption of post), comment (total number of comments including invisible comments) <= This mode takes longer time than mode 0
      • mode 2: In development, aims to return all the above information and specific id of each users that interact with the page's posts

    • Examples: curl "http://wao.nuhula.website/scanpage?uri=https://www.facebook.com/nuhulacoffee&start=1501561354&end=1502425354"
